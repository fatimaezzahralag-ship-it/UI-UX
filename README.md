# UniSmart Dashboard — Redesigned

A full-stack educational platform dashboard with a bold, editorial design aesthetic.

## Features
- **Admin**: System overview, AI optimization engine, timetable management, room management, user management, messaging, settings
- **Teacher**: Dashboard with module declaration form, timetable, online class scheduling, availability grid, messages, profile  
- **Student**: Dashboard with today's schedule, timetable view, module cards, skill exchange marketplace, messages, profile

## Design System
- **Font**: Syne (display/UI) + JetBrains Mono (code/timestamps)
- **Style**: Brutalist/editorial with hard drop shadows, bold borders, high contrast
- **Colors**: Brand teal (#14b8a6) + Ink neutral palette

## Setup
```bash
npm install
npm run dev
```

## Routes
- `/` — Landing page
- `/auth` — Login (demo: select role and click Sign In)
- `/admin` — Admin dashboard
- `/student` — Student dashboard
- `/teacher` — Teacher dashboard
